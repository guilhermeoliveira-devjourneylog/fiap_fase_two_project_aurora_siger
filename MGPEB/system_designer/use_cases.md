# MGPEB - Use Case Diagram

## Atores
- Operador da Missão
- Sistema Autônomo MGPEB
- Sensores Ambientais

## Casos de Uso

### 1. Gerenciar Fila de Pouso
Ator: Sistema MGPEB  
Descrição: Organiza módulos por prioridade e criticidade.

### 2. Monitorar Condições Ambientais
Ator: Sensores  
Descrição: Fornece dados de atmosfera, área e sensores.

### 3. Simular Descida
Ator: Sistema MGPEB  
Descrição: Calcula altura, velocidade e densidade.

### 4. Autorizar Pouso
Ator: Sistema MGPEB  
Descrição: Aplica lógica booleana para decisão.

### 5. Gerenciar Alertas
Ator: Sistema MGPEB  
Descrição: Identifica falhas críticas (combustível, sensores).

### 6. Gerar Relatório
Ator: Sistema MGPEB  
Descrição: Consolida resultados finais.
